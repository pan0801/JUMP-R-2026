## 2.研究问题

本研究基于美国NHANES人群数据，分析不同吸烟状态和成年人高血压患病的相关性，控制
年龄、BMI、性别、种族等协变量，明确吸烟是否为高血压患病的独立危险因素

## 3.数据来源

使用公开可用的NHANES公共数据库，样本量大，变量规范。

## 4.主要变量

-结局变量：高血压患病情况（二分类：患病/未患病）
-暴露变量：吸烟状态
-协变量：性别、种族、BMI（体质指数）、年龄、教育水平、PIR（收入贫困比）
