## 三、变量定义补充说明

### 高血压定义

采用“或”逻辑，满足以下任一条件即定义为高血压（1=有高血压）：

| 条件 | 说明 |
|------|------|
| SBP_avg ≥ 140 mmHg 或 DBP_avg ≥ 90 mmHg | 实测血压升高 |
| BPQ020 == 1 | 曾被医生诊断高血压 |

其中：
- SBP_avg = (BPXS1Y + BPXS2Y + BPXS3Y) / 3（3次测量平均值）
- DBP_avg = (BPXDI1 + BPXDI2 + BPXDI3) / 3（3次测量平均值）

### 吸烟定义

| 分类 | 条件 | 说明 |
|------|------|------|
| Never | SMQ020 == 2 | 一生吸烟 <100 支 |
| Former | SMQ020 == 1 且 SMQ040 == 3 | 吸过 ≥100 支，现在不吸 |
| Current | SMQ020 == 1 且 SMQ040 ∈ {1, 2} | 吸过 ≥100 支，现在每天或偶尔吸 |
